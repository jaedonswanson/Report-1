\section {Statistical Methods}
The first analysis completed was to assess if the treatment an individual received had an impact on blood pressure (BP). This was done by completing an analysis of variance (ANOVA) with BP as the response variable and treatment as the explanatory. After the ANOVA a Tukey-Kramer test was used to compare the groups for differences between specific treatments.

After the ANOVA was completed, potential confounding variables were analyzed for impact on treatment effect. This was done by completing an analysis of covariance (ANCOVA) with sex and age as potential covariates. After the ANCOVA, interactions were assessed to investigate if a persons sex or age had an impact on treatment effect.

All of the data analysis and figure creation was completed in R 4.5.2. The code used for data analysis can be found in the attached appendix.

\section {Results}
The treatment an individual received had a significant effect on their BP ($P < 0.001$, $F_{2, 177} = 35.07$). Persons receiving Medication A had significantly lower BP than those in the placebo group ($P < 0.001$). Individuals in the group receiving medication A also had a significantly lower BP than those receiving medication B ($P < 0.001$) (Fig. 1).

There was no significant impact of a persons sex on treatment effect ($P = 0.0293$, $F_{2, 168} = 1.238$) (Fig. 2). Similarly, age had no significant effect on treatment effect ($P = 0.467$, $F_{2, 168} = 0.764$) (Fig 3). 


\section {Conclusions}
Medication A caused significantly lower BP than both medication B and the placebo group. This result was not affected by the recipients age or sex. For this reason, if a person is looking to lower their BP, medication A should be prescribed in favor of Medication B.

#### Loading Packages and Setting Up Data Frame ####
# Loading packages
library(tidyverse)
library(car)
library(emmeans)
# Creating the data frame that will be used for our analysis
bp <- read_csv("data.csv")
## changing the treatment and sex columns so R sees it as a categorical variable
bp$Treatment <- as.factor(bp$Treatment)
bp$Sex <- as.factor(bp$Sex)

#### ANOVA for Treatment Effect ####
# Fitting a two-way ANOVA model
sex_model <- aov(BP ~ Treatment, data = bp)
summary(sex_model)
## Assumption Check
plot(residuals(sex_model))
abline(0,0)
# Analyzing Treatment Differences
TukeyHSD(sex_model, conf.level = .95)
plot(TukeyHSD(sex_model, conf.level=.95), las = 2)


#### ANCOVA for Interaction(s) ####
# Fitting models to the data
bp_fm <- lm(BP ~ Treatment * Age * Sex, data = bp, 
            contrasts = list(Treatment = contr.sum, Sex = contr.sum))
## Assumption Checks
plot(residuals(bp_fm))
abline(0,0)
# adding the predictions to the data frame
bp$predicted <- predict(bp_fm)
# Testing for significance of interaction.
Anova(bp_fm, type = 3)
